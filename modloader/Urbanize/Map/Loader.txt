PT: Deixe este arquivo dentro do modloader. Você tem opção de apagar ou colocar algo como "#" na linha para o arquivo não ser carregado.
    Para mobile/console: coloque a linha abaixo no final do seu "gta.dat" com um editor de texto, e o arquivo ".ipl" na pasta "data/maps". É necessário aumentar limites do jogo (como o limite de arquivos .ipl, com fastman92 limit adjuster) ou fusionar os arquivos. Procure por tutoriais ou f92la já configurado para isto.

EN: Keep this file inside modloader. You have the option of deleting or putting something like "#" on the line so that the file is not loaded.
    For mobile/console: put the line below at the end of your "gta.dat" with a text editor, and the ".ipl" file in the "data/maps" folder. Is required to increase game limits (as .ipl file limit with fastman92 limit adjuster) or merge the files. Search for tutorials or some f92la already configured for this.
